Overview

WIPERS is a multi-robot autonomous system designed to coordinate multiple robots for simultaneous task execution. The system integrates computer vision for whiteboard detection, motion planning for convergence paths, and real-time communication for task allocation among robots.

Challenge & Problem Statement

The challenge was to develop an autonomous multi-robot system capable of efficiently coordinating to erase magnetic whiteboards simultaneously. Key difficulties included:

  • Coordinating multiple robots without centralized control
  • Implementing robust whiteboard detection and path planning
  • Handling dynamic environments and collision avoidance
  • Ensuring task allocation is fair and efficient across robots

Solution & Approach

We developed an integrated ROS-based architecture that combines:

  • Perception: Computer vision algorithms for whiteboard detection and localization
  • Path Planning: Convergence algorithms ensuring robots meet at optimal points
  • Task Allocation: Distributed task allocation based on robot proximity and availability
  • Control: Real-time motor control and feedback loops for accurate execution

The system uses Python with ROS for middleware communication and MATLAB Simulink for validation and testing.

Results & Outcomes

The system successfully demonstrated:

  • Multi-robot coordination without collision
  • Reliable whiteboard detection and erasure
  • Adaptive behavior in varying lighting and board conditions
  • Scalability to additional robots with minimal reconfiguration

The implementation validated the efficiency of distributed task allocation and established a foundation for future autonomous multi-robot systems.

Key Learnings

  • Distributed algorithms are essential for scalable autonomous systems
  • Robust error handling and recovery mechanisms are critical in real-world deployment
  • Extensive simulation and testing reduce field deployment time
  • Clear communication protocols between robots are fundamental to coordination
  • Graceful degradation—ensuring systems function even when components fail—is as important as nominal operation
