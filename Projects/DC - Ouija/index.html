Overview

Ouija is an interactive robotic art installation reimagining the classic Ouija board using robotics, computer vision, and AI.

Problem Statement

Create an interactive robotic installation blending technology with artistic narrative for engaging real-time visitor interaction.

Solution & Approach

Designed a robotic planchette system with vision-based participant tracking, gesture recognition, and real-time motion control using ROS.

Results & Outcomes

Successfully created an installation that engaged visitors and sparked conversation about technology, art, and human-robot interaction.

Key Learnings

Interactive installations require robust real-time performance and graceful error handling. Robotics has creative applications beyond traditional engineering.
